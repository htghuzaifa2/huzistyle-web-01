# Huzistyle Web

This is a private project source code for **Huzistyle**.

## Features

- 🛍️ **E-commerce Ready**: Complete shopping flow (Cart, Checkout, Products).
- 🔘 **AssistiveTouch**: Smart floating navigation ball with physics-based drag (Framer Motion).
- 📱 **Mobile Optimized**: Fully responsive design for all devices.
- ⚡ **High Performance**: Built with Next.js 15 for speed and SEO.
